# קידוש החודש — תוסף אוצריא

איורים אינטראקטיביים להבנת מהלך גרמי השמיים, כתוסף לאפליקציית [אוצריא](https://github.com/Otzaria/otzaria).

התוסף כולל שלושה איורים:

1. **מופעי הירח** — מערכת שמש–ארץ–ירח במבט-על, עם תצוגת הירח כפי שנראה מכדור הארץ ואחוז ההארה לאורך חודש הלבנה.
2. **מהלך השמש (שנת חמה)** — כדור השמיים מנקודת מבט הצופה: הסיבוב היומי של השמש, נדידת המסלול לאורך השנה, ארבע התקופות (ניסן/תמוז/תשרי/טבת), ועומק השמש בחצות הלילה בכל עונה. כולל סיבוב אופקי של המבט ומעבר ל"עכשיו".
3. **כוכבי הלכת** — מיקום כל כוכבי הלכת בכיפת השמיים לפי תאריך, שעה ומיקום הצופה (חישוב אלגוריתם Paul Schlyter).

## תכונות
- רקע בהיר/כהה לאיורים, בנוסף לאימוץ אוטומטי של ערכת הנושא של אוצריא.
- כפתורי "היום" / "עכשיו" המעבירים את הירח והשמש למצב הנוכחי.
- פועל אופליין לחלוטין — כל הנכסים מוטמעים, ללא תלות ברשת.

## התקנה
הורידו את קובץ `.otzplugin` מעמוד ה-[Releases](../../releases), ובאוצריא:
**כלים → 🧩 תוספים → ⊕ התקן תוסף חדש**, ובחרו את הקובץ.

## בנייה
```bash
otzaria pack-plugin .
```
או מקוד המקור של אוצריא:
```bash
dart run tool/plugins/package_plugin.dart <path-to-this-folder>
```
